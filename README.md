# TP Hooks Avancés – Niveau Débutant

Ce projet React accompagne une vidéo de démonstration des **hooks avancés**.  
Il regroupe plusieurs mini-exemples simples pour découvrir et pratiquer :

- `useReducer`
- `useRef`
- `useEffect` (avec nettoyage)
- un hook personnalisé de type `useFetch`

---

##  Objectifs pédagogiques

- Comprendre quand préférer `useReducer` à `useState`.
- Manipuler des références avec `useRef` (focus, valeur persistante…).
- Observer le cycle de vie d’un composant avec `useEffect`.
- Créer et utiliser un **hook personnalisé** pour factoriser la logique de requêtes HTTP.

---

##  Prérequis

- Node.js installé (version récente recommandée)
- Connaissances de base en :
  - JavaScript (ES6+)
  - React (composants, props, `useState`)

---

## Installation & lancement

```bash
# Cloner ou télécharger le projet
cd projet-hooks-avances

# Installer les dépendances
npm install

# Lancer l'application en mode développement
npm start
```
